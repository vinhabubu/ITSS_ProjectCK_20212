Code-Academyへようこそ

    Code-AcademyはWeb開発者サイトであり、HTML、CSS、C ++、AngularWebプログラミングなどのWeb開発言語に関するチュートリアルとリファレンスがあります。 また、魅力的な外観の簡単な構文を提供します。

    プログラミング言語

    HTML

    HTMLは、Webブラウザで表示するように設計されたドキュメントの標準マークアップ言語です。 カスケードスタイルシートとJavaScriptによって支援できます。

    CSS

    カスケードスタイルシートは、HTMLなどのマークアップ言語で記述されたドキュメントのプレゼンテーションを記述するために使用されるスタイルシート言語です。

    C++

    C ++は、Cプログラミング言語または「CwithClasses」の拡張としてBjarneStroustrupによって作成された汎用プログラミング言語です。

    Angular

    Angularは、Googleと個人や企業のコミュニティが主導するTypeScriptベースのオープンソースWebアプリケーションフレームワークです。
